# DigitalGear

Sitio web profesional de información sobre tecnología, gadgets, reviews, comparativas y tendencias tecnológicas.

## Descripción

DigitalGear es una plataforma de contenido tecnológico diseñada para atraer tráfico orgánico mediante SEO optimizado y generar ingresos a través de Google AdSense y enlaces de afiliados.

## Tecnologías Utilizadas

- **HTML5**: Estructura semántica
- **CSS3**: Diseño responsive y moderno
- **JavaScript**: Interactividad y funcionalidades dinámicas
- **SEO**: Optimización para motores de búsqueda
- **Google AdSense**: Monetización mediante publicidad

## Cómo Desplegar

### Opción 1: Hosting Tradicional
1. Sube todos los archivos a tu servidor web (cPanel, FTP, etc.)
2. Asegúrate de que el dominio apunte correctamente
3. Verifica que todos los enlaces funcionen

### Opción 2: GitHub Pages
1. Crea un repositorio en GitHub
2. Sube todos los archivos
3. Activa GitHub Pages en la configuración del repositorio
4. Tu sitio estará disponible en `username.github.io/digitalgear`

### Opción 3: Vercel / Netlify
1. Conecta tu repositorio de GitHub
2. La plataforma detectará automáticamente los archivos HTML
3. El sitio se desplegará automáticamente

## Configuración de Google AdSense

1. **Solicita aprobación en AdSense**:
   - Ve a https://www.google.com/adsense
   - Registra tu sitio web
   - Espera la aprobación (generalmente 1-2 semanas)

2. **Configura ads.txt**:
   - Edita `assets/ads.txt`
   - Agrega tu Publisher ID de AdSense

3. **Inserta el código de anuncios**:
   - Copia el código de AdSense
   - Pégalo en los espacios `.ad-space` del HTML

## Mantenimiento

### Crear Nuevo Artículo
1. Duplica `articulo.html`
2. Cambia el contenido H1, título, descripción
3. Actualiza las imágenes
4. Agrega el artículo a la página de categoría correspondiente
5. Actualiza `sitemap.xml`

### Agregar Nueva Categoría
1. Duplica `categoria.html`
2. Actualiza el título y contenido
3. Agrega la categoría al menú principal
4. Actualiza `sitemap.xml`
